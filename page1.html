Food Order
It is a web application where anyone can order food online .Various restaurants can use this app for taking orders online. This website is created using tools like Eclipse , Apache Tomcat, MySQL, JSP,Java, JDBC, HTML and CSS. At first the user has to create an account .Then the user will be eligible for log in. then a new page opens where the user can see the restuarent menu . Then the user can selects the items to order .After selecting the items to order it goes to the cart .Then the user has to give his/her delivery address . After that users bill is to be displayed .Then the users order will be delivered in next 10 minutes

Desktop Application
Med Pharamacy
It is a standalone applications specially designed for pharamacists to manage record of medicines . This application is created using eclipse , ApacheTomcat, Mysql, Java ,JDBC ,Swing , awt . At first the admin will create an account for the user . After registering the user will login .User can add medicine,update medicine,delete medicine,create bill etc . User can see his total sale per day. The admin can create other admins, admins can also update ,delete anything. Admin can See the daily sale of users, monthly sale, can see highest selling medicine of the month, expiry dates of all medicines and can also see all customers information.
